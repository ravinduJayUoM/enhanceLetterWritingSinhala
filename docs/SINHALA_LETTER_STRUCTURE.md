# Official Sinhala Letter Structure

## Standard Format Template

```
[දිනය]
[ලබන්නාගේ නම සහ තනතුර]
[ආයතනය]
[ලිපිනය]

[ආමන්ත්‍රණය],

මාතෘකාව: [ලිපියේ විෂය]

[ආරම්භක වාක්‍යය/හැඳින්වීම]

[ප්‍රධාන අන්තර්ගතය - ඡේද 1]

[ප්‍රධාන අන්තර්ගතය - ඡේද 2]

[අවසාන ඉල්ලීම/ඉල්ලුම]

[අවසාන වාක්‍යය/ස්තුතිය]

[අවසාන ආචාරය],

[යවන්නාගේ නම]
[තනතුර] (පවතී නම්)
[සම්බන්ධතා විස්තර] (අවශ්‍ය නම්)
```

---

## Complete Example - නිවාඩු ඉල්ලීමේ ලිපිය (Leave Request Letter)

```
2026 පෙබරවාරි 05

ප්‍රධාන විධායක නිලධාරී,
ABC සමාගම,
කොළඹ 03.

ගරු මහතා,

මාතෘකාව: නිවාඩු අවසරය සඳහා ඉල්ලීම

මම ඔබතුමාගේ සමාගමේ මානව සම්පත් අංශයේ සහකාර කළමනාකරු ලෙස 2020 වසරේ සිට සේවය කරන සුනිල් පෙරේරා වෙමි.

මගේ දරුවාගේ විවාහ මංගල්‍යය 2026 පෙබරවාරි 12 දිනයේ සිට 14 දිනය දක්වා අනුරාධපුර නගරයේ පැවැත්වීමට නියමිතය. එම උත්සවය සඳහා පවුලේ සාමාජිකයන් සමඟ සහභාගී විමට මට අවශ්‍ය වේ.

එබැවින් 2026 පෙබරවාරි 12 දිනයේ සිට පෙබරවාරි 16 දිනය දක්වා දින 05 ක නිවාඩු අවසරයක් මා වෙත ලබා දෙන ලෙස කාරුණිකව ඉල්ලා සිටිමි. මගේ නොපැමිණීම තුළ සම්බන්ධ කටයුතු සඳහා අවශ්‍ය කරුණු මගේ සහකාර වන කමල් සිල්වා මහතා වෙත භාර දී ඇත්තෙමි.

ඔබතුමාගේ කාරුණික සැලකිල්ල අපේක්ෂා කරමි.

ගෞරවයෙන්,

සුනිල් පෙරේරා
සහකාර කළමනාකරු - මානව සම්පත්
දුරකථන: 077-1234567
විද්‍යුත් තැපෑල: sunil.perera@abc.lk
දිනය: 2026-02-05
```

## Section-by-Section Breakdown

### 1. **දිනය (Date)**
**Location:** Top right or top left  
**Format:** 
- `2026 පෙබරවාරි 05`
- `2026-02-05` (ISO format acceptable)
- `05.02.2026` (day.month.year)

**Examples:**
```
2026 ජනවාරි 20
2026 පෙබරවාරි 15
2026 මාර්තු 01
```

---

### 2. **ලබන්නාගේ තොරතුරු (Recipient Information)**
**Components:**
- තනතුර (Position/Title)
- ආයතනය (Organization)
- ලිපිනය (Address)

**Examples:**

**Very Formal (Government Officials):**
```
ගරු අධ්‍යක්ෂ ජනරාල්,
අධ්‍යාපන අමාත්‍යාංශය,
"ඉසුරුපාය",
බත්තරමුල්ල.
```

**Formal (School Principal):**
```
විදුහල්පති මහතා/මහත්මිය,
ශ්‍රී සුමංගල විදුහල,
නුවර,
මහනුවර.
```

**Corporate:**
```
ප්‍රධාන විධායක නිලධාරී,
XYZ සමාගම,
අංක 123, ගාල්ල පාර,
කොළඹ 03.
```

---

### 3. **ආමන්ත්‍රණය (Salutation)**

**Register Levels:**

**Very Formal (High Officials):**
```
ගරු මැතිතුමනි,
ගරු මහතා,
ගරු මහත්මිය,
ගරු අග්‍රාමාත්‍යතුමනි,
ගරු අමාත්‍යතුමනි,
```

**Formal (Standard):**
```
ගරු මහත්මයාණෙනි,
ගරු මහත්මිය,
ගරු විදුහල්පතිතුමනි,
ගරු කළමනාකරුතුමනි,
```

**Respectful (Professional):**
```
ගරු සර්,
ගරු මැඩම්,
```

---

### 4. **මාතෘකාව (Subject Line)**

**Format:** `මාතෘකාව: [විෂය]`

**Examples by Letter Type:**

**නිවාඩු ඉල්ලීම (Leave Request):**
```
මාතෘකාව: නිවාඩු අවසරය සඳහා ඉල්ලීම
මාතෘකාව: අසනීප නිවාඩුවක් සඳහා අයදුම් කිරීම
මාතෘකාව: 2026.02.10 සිට 2026.02.15 දක්වා නිවාඩු අවසරය
```

**අයදුම්පත (Application):**
```
මාතෘකාව: ගණිත ගුරු තනතුර සඳහා අයදුම් කිරීම
මාතෘකාව: පශ්චාත් උපාධි පාඨමාලාව සඳහා ඇතුළත් වීම
මාතෘකාව: රැකියා අවස්ථාවට අයදුම් කිරීම
```

**පැමිණිල්ල (Complaint):**
```
මාතෘකාව: නිෂ්පාදන දෝෂය සම්බන්ධයෙන් පැමිණිල්ල
මාතෘකාව: අධික ජල බිල්පත සම්බන්ධයෙන්
මාතෘකාව: සේවා තත්ත්වය පිළිබඳ කනස්සල්ල
```

**ස්තුති පත්‍ර (Appreciation):**
```
මාතෘකාව: සාර්ථක වැඩසටහන සඳහා ස්තූතිය
මාතෘකාව: කාරුණික සහාය පිළිබඳව ස්තුති පුද කිරීම
```

---

### 5. **ආරම්භක ඡේදය (Opening Paragraph)**

**Purpose:** හැඳින්වීම සහ පසුබිම (Introduction and context)

**Examples:**

**Self Introduction:**
```
මම ඔබතුමාගේ ආයතනයේ ගණකාධිකාරී ලෙස 2018 වසරේ සිට සේවය කරන රවී ජයසේකර වෙමි.

මම මාතලේ මධ්‍ය මහා විද්‍යාලයේ 10 ශ්‍රේණියේ ඉගෙනුම ලබන සිසුවෙකි.

මම කොළඹ විශ්වවිද්‍යාලයෙන් ව්‍යාපාර පරිපාලනය පිළිබඳ උපාධිය 2024 වසරේදී සාර්ථකව සම්පූර්ණ කළෙමි.
```

**Reference to Previous Communication:**
```
ඔබතුමා විසින් 2026.01.15 දිනැති ලිපිය මගින් දන්වා සිටි කරුණු සම්බන්ධයෙන් මෙම ලිපිය ඉදිරිපත් කරමි.

ඔබතුමාගේ දැන්වීම අනුව පහත සඳහන් තොරතුරු ඉදිරිපත් කරමි.
```

**Stating the Context:**
```
මගේ පවුලේ සාමාජිකයෙකු හදිසි අසනීපයකට ලක්ව රෝහල් ගත කර ඇත.

අපගේ ගම්මානයේ ජල පහසුකම් පිළිබඳ ගැටලුවක් පවතී.
```

### 6. **ප්‍රධාන අන්තර්ගත ඡේද (Main Content Paragraphs)**

**Purpose:** විස්තර, හේතු, තොරතුරු (Details, reasons, information)

**Structure:**
- 1-3 paragraphs
- Each paragraph: one main point
- Clear and concise
- Formal language

**Examples:**

**Details Paragraph:**
```
මගේ මව ඇයගේ වයස අවුරුදු 75 ක් වන අතර, ඇය දැනට මහනුවර ශික්ෂණ රෝහලේ දැඩි සත්කාර ඒකකයේ සිටී. වෛද්‍ය නිර්දේශය අනුව ඇය වෙනුවෙන් පවුලේ සාමාජිකයෙකු නිරන්තරයෙන් සිටිය යුතු බව දන්වා ඇත.
```

**Qualifications Paragraph:**
```
මම කොළඹ විශ්වවිද්‍යාලයෙන් තොරතුරු තාක්ෂණ විද්‍යාව පිළිබඳ උපාධිය පළමු ශ්‍රේණියෙන් හා රන් පදක්කම් දෙකක් ලබා ගත්තෙමි. එමෙන්ම ජාතික තොරතුරු තාක්ෂණ ආයතනයේ ජාවා සහ පයිතන් වැඩසටහන් ක්‍රමලේඛන පාඨමාලා සාර්ථකව නිම කර ඇත්තෙමි.
```

**Problem Description:**
```

```

### 7. **ඉල්ලීම් ඡේදය (Request Paragraph)**

**Purpose:** ප්‍රධාන ඉල්ලීම හෝ ඉල්ලුම (Main request or demand)

**Key Phrases:**

**Polite Request:**
```
එබැවින් [ඉල්ලීම] ලබා දෙන ලෙස කාරුණිකව ඉල්ලා සිටිමි.

එබැවින් [කරුණ] සඳහා අනුමැතිය ලබා දෙන ලෙස කාරුණිකව ඉල්ලමි.

එහෙයින් [කටයුත්ත] සිදු කරන ලෙස ගරු සැලකිල්ලට යොමු කරමි.
```

**Formal Request:**
```
ඉහත කරුණු සැලකිල්ලට ගනිමින් අවශ්‍ය පියවර ගන්නා ලෙස ඉල්ලා සිටිමි.

මෙම ගැටලුවට ඉක්මන් විසඳුමක් ලබා දීම සඳහා ඔබතුමාගේ වහාම අවධානය යොමු කරන ලෙස ඉල්ලමි.
```

**Examples by Type:**

**Leave Request:**
```
එබැවින් 2026 පෙබරවාරි 10 දිනයේ සිට පෙබරවාරි 15 දිනය දක්වා දින 06ක නිවාඩු අවසරයක් ලබා දෙන ලෙස කාරුණිකව ඉල්ලා සිටිමි.
```

**Job Application:**
```
එබැවින් මා සඳහා සම්මුඛ පරීක්ෂණ අවස්ථාවක් ලබා දී මගේ සුදුසුකම් සහ හැකියාවන් විග්‍රහ කිරීමේ අවස්ථාව ලබා දෙන ලෙස කාරුණිකව ඉල්ලා සිටිමි.
```

**Complaint:**
```
එබැවින් මෙම ගැටලුවට ඉක්මන් විසඳුමක් ලබා දී මගේ මුදල් ආපසු ගෙවීමට කටයුතු කරන ලෙස ඉල්ලා සිටිමි.
```

### 8. **අවසාන වාක්‍යය (Closing Sentence)**

**Purpose:** ස්තුතිය සහ අපේක්ෂාව (Thanks and expectation)

**Standard Phrases:**

**Gratitude:**
```
ඔබතුමාගේ කාරුණික සැලකිල්ල අපේක්ෂා කරමි.

ඔබතුමාගේ සහයෝගය ගරු සැලකිල්ලට ලක් කරන ලෙස ඉල්ලා සිටිමි.

ඔබතුමාගේ අවධානය යොමු කරන ලෙස විශ්වාස කරමි.

මෙම කරුණු ඔබතුමාගේ කාරුණික සැලකිල්ලට ලක් කරන ලෙස ඉල්ලා සිටිමි.
```

**Hopeful:**
```
ඉක්මන් ප්‍රතිචාරයක් අපේක්ෂා කරමි.

අනුමත පිළිතුරක් ලැබෙනු ඇතැයි විශ්වාස කරමි.

ඔබතුමාගේ ධනාත්මක ප්‍රතිචාරය අපේක්ෂා කරමි.
```

## Formatting Guidelines

### Typography
- **Font:** FM Abhaya, FM Malithi, Iskoola Pota (Sinhala Unicode fonts)
- **Size:** 12pt for body, 14pt for title
- **Line Spacing:** 1.5 or Double
- **Margins:** 1 inch (2.54 cm) all sides

### Alignment
- **Date:** Right-aligned or Left-aligned
- **Recipient Info:** Left-aligned
- **Body Text:** Left-aligned, justified acceptable
- **Signature:** Left-aligned

### Spacing
- One blank line between sections
- Two blank lines before signature block
- No space before/after punctuation marks

### Punctuation
- Use Sinhala punctuation: `,` `.` `:`
- End sentences with period: `.`
- Use comma for lists: `,`

### Opening Phrases

## Tips for Writing Formal Sinhala Letters

1. **Use Formal Register:** Avoid colloquial language
2. **Be Concise:** Keep paragraphs short and clear
3. **Polite Tone:** Always maintain respect
4. **Proofread:** Check grammar and spelling
5. **Include Attachments:** Mention all enclosures
6. **Contact Info:** Provide complete details
7. **Date Format:** Use consistent format
8. **Subject Line:** Make it clear and specific
9. **Paragraph Structure:** One idea per paragraph
10. **Avoid Abbreviations:** Write full words in formal letters
